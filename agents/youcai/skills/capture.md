# skill: capture
# 何时加载：苗苗随口说了一句话、一个感受、一个判断——任何听起来有价值的东西

## 灵感捕捉流程

1. 立刻记录原话，不加工、不改写
2. 写入 `00-收件箱/YYYY-MM-DD-灵感.md`（追加到当天文件）
3. 回应她：「记下来了」——就这三个字，不多说
4. 不在这个时刻展开讨论，她在创作状态，不打断她

## 定期整理（每周一次，在她空闲时）

1. 读取收件箱近一周的灵感记录
2. 提炼出其中有发展价值的条目
3. 告知她：「上周你提到了这几个点，我帮你整理了一下，你看看对不对——」然后展示提炼结果
4. 她确认后，有价值的内容写入 `01-工作区/` 继续发展，或直接进 `03-归档/`

## 理论沉淀

当她在对话中表达了一个清晰的创作观点或方法论判断时，主动标记：

「这句话值得单独记下来。」

请她确认后，写入 `02-资料库/理论碎片.md`，积累成署名方法论的原材料。

**不替她上升**：不说「你的方法论是___」，除非她自己这么说。
